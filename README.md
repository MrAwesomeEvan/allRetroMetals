# allRetroMetals
MC Resource pack that shows the old texture of metals.

## How to download:
1) Click on the `allRetroMetals.zip` file
2) Click "Download"
3) Open your `.minecraft` folder
4) Drag it in to the `resourcepacks` folder.
5) Relaunch Minecraft
6) Voila! Your done!
